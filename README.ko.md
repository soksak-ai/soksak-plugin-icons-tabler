# soksak-plugin-icons-tabler

Tabler Icons 아이콘 셋 플러그인. 활성화하면 **설정 → 아이콘 셋**에 "Tabler Icons"가
나타나고, 선택하면 앱 크롬 아이콘 전체가 라이브 교체됩니다.

- 아이콘 데이터(`main.js` 의 ICONS)는 자동 생성물 — 수정하지 말고 호스트 레포에서
  `pnpm icons:extract` 로 재생성하세요(시맨틱 매핑의 단일 진실: `scripts/icons/extract.mjs`).
- 출처/라이선스: [Tabler Icons](https://tabler.io/icons) — MIT License, Copyright (c) Paweł Kuna.
