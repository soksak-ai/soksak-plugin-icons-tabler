# soksak-plugin-icons-tabler

Tabler Icons icon set plugin. When activated, **"Tabler Icons"** appears under **Settings →
Icon Set**, and selecting it live-swaps all app chrome icons.

- The icon data (`ICONS` in `main.js`) is auto-generated — do not edit it manually. Regenerate
  it from the host repo with `pnpm icons:extract` (single source of truth for semantic
  mapping: `scripts/icons/extract.mjs`).
